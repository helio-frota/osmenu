# osmenu
An interactive menu to show information about the system,
to show features from OS and Readline node modules.

> This App demo was part of [NUGCE](http://nugce.org/) talk:

> http://nugce.org/5o-meetup-nugce/

> 26 November 2016


run:

```
node Readline.js
```
